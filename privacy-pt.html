Política de Privacidade

Última atualização: julho de 2025

O aplicativo “Simple Spirit Level” não coleta dados pessoais. Algumas informações técnicas (como modelo do dispositivo, versão do sistema operacional) podem ser coletadas automaticamente pela plataforma (por exemplo, Google Play ou o sistema operacional).

1. Publicidade (opcional)

O aplicativo pode futuramente incluir anúncios, como banners ou vídeos publicitários. Nesse caso, terceiros como o Google AdMob podem coletar informações sobre o seu dispositivo e uso para exibir anúncios personalizados.

A coleta de dados por esses provedores está sujeita às respectivas políticas de privacidade.

2. Sem registro

Não é necessário se registrar ou inserir dados pessoais para usar este aplicativo.

3. Nenhum armazenamento de dados do usuário

O aplicativo não armazena dados pessoais no dispositivo ou na nuvem.
